Analyzing Public Sentiment from Facebook Pages using AI

What is CivicPulse?

CivicPulse is a data analytics platform designed to measure public and political sentiment from publicly available Facebook Page posts, comments, and reactions. Using AI-powered sentiment analysis, it visualizes how citizens engage with civic discussions online.

How It Works

  1. Connects to public Facebook Pages (e.g., political leaders, news outlets)
  2. Fetches posts, comments, and reactions using Facebook Graph API
  3. Analyzes tone and engagement level (positive, negative, neutral)
  4. Displays results through a clean and interactive dashboard
